run ( )

Traite les données envoyées par le visiteur via le formulaire de connexion (variables $_POST['login'] et $_POST['password']). Le mot de passe est vérifié en utilisant les méthodes de la classe Database. Si le mot de passe n'est pas correct, on affiche le message "Pseudo ou mot de passe incorrect." Si la vérification est réussie, le pseudo est affecté à la variable de session.
